¿Por qué un lenguaje de programación sólo puede utilizarse en algunos sistemas operativos y en otros no?.
No todos los lenguajes de programación tienen una máquina virtual que permita abstraer y hacer interoperable entre los distintos sistemas operativos. por ejemplo Java con su máquina virtual permite ejecutarse en diversos sistemas operativos.

¿Qué tipo de máquina virtual es virtualBox?
Es un software de virtualización para arquitecturas x86/amd64. Se utiliza para generar virtualizaciones de sistemas operativos dentro de un ordenador físico existente, generando lo que se conoce como máquinas virtuales.

Si tengo más de una máquina virtual instalada, y una se rompe, ¿esto afecta a las demás? ¿por qué? 
No, porque cada máquina virtual representa una participación de recursos individual, cada ambiente o sistema operativo está por separado.
